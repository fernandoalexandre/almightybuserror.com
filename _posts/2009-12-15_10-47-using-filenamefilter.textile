To used a _FilenameFile_ you need to implement a method _accept_ which has as arguments the directory when the file exists and the name of the file. It is normally used in conjuction with the _list_ method from _File_ object.

The following example prints all the files with extension "jpg" in a certain directory:

<pre><code class="prettyprint lang-java">public static void main(String[] args) {
    FilenameFilter filter = new FilenameFilter() {
        @Override
        public boolean accept(File dir, String name) {
            return name.endsWith(".jpg");
        }
    };

    for(String c : args) {
        File dir = new File(c);

        if(dir.isDirectory()) {
            System.out.println("Directory: " + c);
            for(String curr : dir.list(filter))
                System.out.println(curr);
        }
    }
}</code></pre>
